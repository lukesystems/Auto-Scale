- Projects CRUD
- Product brief (manual + AI)
- Competitors + source posts
- TrendWatch run + signal scoring + structured insights

- Channel sync
- Calendar view
- Bulk schedule with gap-aware spacing

- Patterns to repeat / kill
- Next-week strategy auto-built from winners
